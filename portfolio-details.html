A web-based application that allows Ghanaian tertiary students to apply for, manage, and track their educational loans provided by the government of Ghana.

Company: Students Loan Trust Fund
Main Features:
  • Requirements GatheringI
  • Online students loan application. Loans are applied online where student loans amounts are computed automatically based on means testing methodology. The means testing methodolgy computes students' loans based on their socio economic state
  • Loan eligibility check. System checks applicants eligibility automatically using their Ghana card details connecting to the API of the National Identification Authority (NIA)
  • Document management. Allows applicants to upload supporting documents such as Ghana Card, School ID, School admission letters, passport picture and any other documents to support their applications. All uploaded documents are scanned in the background against malwares
  • Supports electronic signatures and finger prints. The finger prints are checked against NIAs API to validate users
  • Live application tracking. Applicants are able to check the status of their applications anytime through their online portal
  • It has a mobile version on Android and IOS
  • Loan repayment functionality
  • Personal details amendment
  • Loan statements (detail & summary)
  • 2 factor authentication to verify phone number during account creation
Tools Used:
  • HTML, CSS, Javascript for the UI
  • DotNet Core and C# for the backend REST API
  • Microsoft SQL for the database
  • Flutter for the mobile app
Roles Played:
  • Created the API project structure and the application database
  • Supervised the UI and mobile version
  • Supervised other programmers in the project
  • Participated in standups during the project process
